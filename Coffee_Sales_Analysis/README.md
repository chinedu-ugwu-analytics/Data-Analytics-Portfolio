# Coffee Sales Analysis

## Problem Statement
This project analyzes coffee shop sales data to understand customer purchasing behavior, popular coffee types, and revenue trends across different times of the day.

## Tools Used
- Excel
- Advanced Excel formulas
- Pivot tables

## Dataset Description
The dataset contains coffee sales transactions with details such as date, time of day, coffee type, price, and customer income level.

## Steps Performed
- Cleaned and prepared the dataset
- Analyzed sales by coffee type and income level
- Evaluated revenue patterns by time of day
- Built dashboards to visualize key insights

## Key Insights
- Certain coffee types generated higher revenue consistently
- Morning hours recorded the highest number of transactions
- Higher-income customers preferred premium coffee options

## Conclusion
The analysis provides actionable insights that can help coffee shop owners optimize product offerings, pricing strategies, and staffing based on customer behavior and peak sales periods.
