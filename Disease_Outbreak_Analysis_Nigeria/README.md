# Disease Outbreak Analysis in Nigeria

## Project Overview
This project analyzes disease outbreak data in Nigeria to identify patterns, high-risk regions, and trends that can support public health decision-making.

## Problem Statement
Health authorities need data-driven insights to understand disease spread, regional impact, and outbreak frequency in order to improve response strategies and resource allocation.

## Tools Used
- Power BI
- Excel (Data Preparation)

## Dataset Description
The dataset includes information on:
- Disease type
- Location (State/Region)
- Reported cases
- Time period
- Outbreak severity indicators

## Steps Performed
- Data cleaning and transformation
- Creation of calculated measures
- Analysis of outbreak trends by region
- Visualization using interactive Power BI dashboards

## Key Insights
- Certain regions consistently record higher outbreak cases
- Seasonal patterns affect outbreak frequency
- Rapid response areas show lower severity over time

## Conclusion
The analysis highlights the importance of regional monitoring and timely intervention in managing disease outbreaks across Nigeria.
